## 앱 설명 (4000자 이내)

Gipet은 당신의 GitHub 기여 스트릭을 함께 지켜주는 귀여운 데스크톱 펫입니다.

**주요 기능**

• 데스크톱 펫 — 거위, 강아지, 병아리 등 다양한 캐릭터가 화면 위를 자유롭게 걸어다닙니다
• GitHub 스트릭 추적 — 오늘 커밋을 했는지 메뉴바에서 한눈에 확인
• 커밋 알림 — 오늘 아직 커밋하지 않으면 펫이 찾아와 상기시켜 줍니다
• 7일 기여 그래프 — 메뉴바 아이콘에 최근 7일간의 기여 현황을 색상으로 표시
• 아빠 개그 — 귀여운 말풍선으로 작업 중 잠깐의 웃음을 선사합니다
• 캐릭터 스킨 전환 — 여러 캐릭터 중 마음에 드는 것을 선택하세요

**GitHub 연동**
GitHub 계정으로 로그인하면 공개/비공개 기여 데이터를 불러옵니다. 로그인 없이 GitHub 사용자명만 입력해도 공개 기여 현황을 확인할 수 있습니다.

**프라이버시 우선**
모든 데이터는 기기 내에만 저장됩니다. GitHub 토큰은 macOS 키체인에 안전하게 보관됩니다. 어떠한 분석 데이터도 수집하지 않습니다.

코딩 습관을 만들고 싶은 개발자라면, Gipet이 오늘도 함께합니다. 🐾
