Gipet — Privacy Policy

Last updated: June 24, 2026

1. Overview

Gipet is a macOS menu-bar app that displays a desktop pet character and shows your GitHub contribution streak. We take your privacy seriously. This policy describes what data Gipet accesses and how it is handled.

2. Data We Collect

GitHub Account Data

If you sign in with GitHub, Gipet requests the read:user scope to identify your username and read your public contribution graph. Your GitHub access token is stored securely in the macOS Keychain on your device and is never transmitted to any server other than GitHub's official API (api.github.com).

Contribution Data

Gipet fetches contribution data (commit counts, streak information) directly from GitHub's API. This data is cached locally in memory only and is not stored on disk beyond what GitHub itself provides.

AI Features

Gipet may use an AI API (OpenRouter / OpenAI) to generate speech-bubble messages. If you provide an API key, it is stored only in your local environment and is sent only to the respective AI provider's servers. No personal data is included in AI requests.

3. Data We Do NOT Collect

  • We do not collect any analytics or usage data.
  • We do not share your information with third parties.
  • We do not store any data on remote servers operated by Gipet.

4. Data Storage

  • GitHub token: Stored in the macOS Keychain (encrypted, device-local).
  • GitHub username / preferences: Stored in UserDefaults (device-local).
  • Memes and Notes: Stored in ~/Library/Application Support/Gipet/ (device-local).

5. Network Requests

Gipet makes network requests only to:

  • api.github.com — contribution data and user info
  • github.com — OAuth login flow
  • openrouter.ai or api.openai.com — AI-generated messages (only if an API key is configured)

6. Children's Privacy

Gipet is not directed at children under 13 and we do not knowingly collect data from children.

7. Changes to This Policy

We may update this policy occasionally. Continued use of Gipet after changes constitutes acceptance of the revised policy.
